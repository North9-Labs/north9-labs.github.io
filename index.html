Seam Stable

Post-quantum encrypted transport over UDP. Hybrid Noise_XX + ML-KEM-768 handshake. Replaces scp, netcat, and ssh -L. 247 µs handshake. 568 MiB/s encrypted throughput per core.

$ curl -fsSL https://install.north9.org/seam.sh | sh

Seamless Stable

Post-quantum reverse tunnels — expose any service through a relay you control. HTTP by subdomain, raw TCP by port. Client dials out through NAT with no port forwarding. Built on Seam — every byte is post-quantum encrypted end-to-end.

$ curl -fsSL https://install.north9.org/seamless.sh | sh

Fob Beta

Encrypted vault on any USB drive — passwords, TOTP, SSH keys, secure notes. Argon2id + XChaCha20-Poly1305. Plausible deniability. No install required on the host.

$ curl -fsSL https://install.north9.org/fob.sh | sh

Signet SDK

Cryptographic photo watermarking SDK. Camera apps embed an unforgeable drand beacon proof at the moment the shutter fires — before encoding. Verification is binary and decentralized. iOS, Android, and C/C++ via FFI.
