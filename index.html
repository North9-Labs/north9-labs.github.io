Seam — Post-Quantum Transport

Seam is a secure UDP transport library and CLI designed to replace SSH tunnels for file transfer, port forwarding, and custom protocol work — with ML-KEM-768 post-quantum key exchange baked in from day one.

Crypto

  • ML-KEM-768 + X25519 hybrid KEM
  • Noise_XX authenticated handshake
  • ChaCha20-Poly1305 per-packet AEAD
  • BLAKE3 key derivation
  • 0-RTT session resumption
  • Header protection (QUIC-style)

Transport

  • Multi-stream over single UDP socket
  • Reliable streams + unreliable datagrams
  • CUBIC / BBRv1 congestion control
  • Forward error correction (Reed-Solomon)
  • Keepalive Ping/Pong (non-ACK-eliciting)
  • Path MTU probing

CLI commands

  • seam cp — push & pull with --resume
  • seam connect — interactive shell tunnel
  • seam send / recv — pipe files
  • seam ls — remote directory listing
  • seam config — manage identity & peers
  • seam doctor — diagnose setup
